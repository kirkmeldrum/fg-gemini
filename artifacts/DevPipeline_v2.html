AI Development Session Protocol

Follow this protocol for EVERY development session with Claude, Claude Code, or Windsurf.

Step 1: Read PROJECT_STATUS.md
Say: "Let's continue [Project]. Read PROJECT_STATUS.md"
~1 min
Step 2: Verify Filesystem Access
Confirm access to project directory and review relevant files
~1 min
Step 3: Collaborative Development
Ask questions → Understand patterns → Make changes via filesystem MCP
Main work
Step 4: Quick Validation
1–2 things to verify. Don't over-engineer testing.
~5 min
Step 5: Git Commit
git add -A && git commit -m "descriptive message"
~1 min
